# OM Techno Tools (OMTT) - Landing Page

A modern, high-performance landing page for OM Techno Tools, a precision engineering partner delivering excellence in plastic injection moulds, rubber moulds, and precision fixtures.

## 🚀 Features

- **Modern UI/UX**: Designed with a sleek, industrial aesthetic combined with modern web design principles (glassmorphism, smooth animations).
- **Responsive Design**: Fully optimized for all screen sizes (mobile, tablet, desktop).
- **Interactive Elements**: Features dynamic animations using Framer Motion and a responsive bento-grid layout for services.
- **Theme Support**: Built with Tailwind CSS, supporting dark/light mode configurations.
- **Performance Optimized**: Built with React and Vite for fast loading and development.

## 🛠️ Technology Stack

- **Frontend Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Routing**: Wouter

## 💻 Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
